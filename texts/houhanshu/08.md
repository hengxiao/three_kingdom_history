<a id="hhs.8.p1"></a>
孝靈皇帝諱宏，肅宗玄孫也。曾祖河閒孝王開，祖淑，父萇。世封解瀆亭侯，帝襲侯爵。母董夫人。桓帝崩，無子，皇太后與父城門校尉竇武定策禁中，使守光祿大夫劉儵持節，將左右羽林至河閒奉迎。

<a id="hhs.8.p2"></a>
建寧元年春正月壬午，城門校尉竇武爲大將軍。己亥，帝到夏門亭，使竇武持節，以王青蓋車迎入殿中。庚子，即皇帝位，年十二。改元建寧。以前太尉陳蕃爲太傅，與竇武及司徒胡廣參錄尚書事。

<a id="hhs.8.p3"></a>
使護羌校尉段熲討先零羌。

<a id="hhs.8.p4"></a>
二月辛酉，葬孝桓皇帝于宣陵，廟曰威宗。

<a id="hhs.8.p5"></a>
庚午，謁高廟。辛未，謁世祖廟。大赦天下。賜民爵及帛各有差。

<a id="hhs.8.p6"></a>
段熲大破先零羌於逢義山。

<a id="hhs.8.p7"></a>
閏月甲午，追尊皇祖爲孝元皇，夫人夏氏爲孝元皇后，考爲孝仁皇，夫人董氏爲慎園貴人。

<a id="hhs.8.p8"></a>
夏四月戊辰，太尉周景薨。司空宣酆免，長樂衞尉王暢爲司空。

<a id="hhs.8.p9"></a>
五月丁未朔，日有食之。詔公卿以下各上封事，及郡國守相舉有道之士各一人；又故刺史、二千石清高有遺惠，爲衆所歸者，皆詣公車。

<a id="hhs.8.p10"></a>
太中大夫劉矩爲太尉。

<a id="hhs.8.p11"></a>
六月，京師雨水。

<a id="hhs.8.p12"></a>
秋七月，破羌將軍段熲復破先零羌於涇陽。

<a id="hhs.8.p13"></a>
八月，司空王暢免，宗正劉寵爲司空。

<a id="hhs.8.p14"></a>
九月丁亥，中常侍曹節矯詔誅太傅陳蕃、大將軍竇武及尚書令尹勳、侍中劉瑜、屯騎校尉馮述，皆夷其族。皇太后遷于南宮。司徒胡廣爲太傅，錄尚書事。司空劉寵爲司徒，大鴻臚許栩爲司空。

<a id="hhs.8.p15"></a>
冬十月甲辰晦，日有食之。令天下繫囚罪未決入縑贖，各有差。

<a id="hhs.8.p16"></a>
十一月，太尉劉矩免，太僕沛國聞人襲爲太尉。

<a id="hhs.8.p17"></a>
十二月，鮮卑及濊貊寇幽并二州。

<a id="hhs.8.p18"></a>
二年春正月丁丑，大赦天下。

<a id="hhs.8.p19"></a>
三月乙巳，尊慎園董貴人爲孝仁皇后。

<a id="hhs.8.p20"></a>
夏四月癸巳，大風，雨雹。詔公卿以下各上封事。

<a id="hhs.8.p21"></a>
五月，太尉聞人襲罷，司空許栩免。六月，司徒劉寵爲太尉，太常許訓爲司徒，太僕長沙劉囂爲司空。

<a id="hhs.8.p22"></a>
秋七月，破羌將軍段熲大破先零羌於射虎塞外谷，東羌悉平。

<a id="hhs.8.p23"></a>
九月，江夏蠻叛，州郡討平之。

<a id="hhs.8.p24"></a>
丹陽山越賊圍太守陳夤，夤擊破之。

<a id="hhs.8.p25"></a>
冬十月丁亥，中常侍侯覽諷有司奏前司空虞放、太僕杜密、長樂少府李膺、司隷校尉朱瑀、潁川太守巴肅、沛相荀翌、河內太守魏朗、山陽太守翟超皆爲鉤黨，下獄，死者百餘人，妻子徙邊，諸附從者錮及五屬。制詔州郡大舉鉤黨，於是天下豪桀及儒學行義者，一切結爲黨人。

<a id="hhs.8.p26"></a>
庚子晦，日有食之。

<a id="hhs.8.p27"></a>
十一月，太尉劉寵免，太僕郭禧爲太尉。

<a id="hhs.8.p28"></a>
鮮卑寇并州。

<a id="hhs.8.p29"></a>
是歲，長樂太僕曹節爲車騎將軍，百餘日罷。

<a id="hhs.8.p30"></a>
三年春正月，河內人婦食夫，河南人夫食婦。

<a id="hhs.8.p31"></a>
三月丙寅晦，日有食之。

<a id="hhs.8.p32"></a>
夏四月，太尉郭禧罷，太中大夫聞人襲爲太尉。秋七月，司空劉囂罷。八月，大鴻臚橋玄爲司空。

<a id="hhs.8.p33"></a>
九月，執金吾董寵下獄死。

<a id="hhs.8.p34"></a>
冬，濟南賊起，攻東平陵。

<a id="hhs.8.p35"></a>
鬱林烏滸民相率內屬。

<a id="hhs.8.p36"></a>
四年春正月甲子，帝加元服，大赦天下。賜公卿以下各有差，唯黨人不赦。

<a id="hhs.8.p37"></a>
二月癸卯，地震，海水溢，河水清。

<a id="hhs.8.p38"></a>
三月辛酉朔，日有食之。

<a id="hhs.8.p39"></a>
太尉聞人襲免，太僕李咸爲太尉。

<a id="hhs.8.p40"></a>
詔公卿至六百石各上封事。

<a id="hhs.8.p41"></a>
大疫，使中謁者巡行致醫藥。

<a id="hhs.8.p42"></a>
司徒許訓免，司空橋玄爲司徒。夏四月，太常來豔爲司空。

<a id="hhs.8.p43"></a>
五月，河東地裂，雨雹，山水暴出。

<a id="hhs.8.p44"></a>
秋七月，司空來豔免。

<a id="hhs.8.p45"></a>
癸丑，立貴人宋氏爲皇后。

<a id="hhs.8.p46"></a>
司徒橋玄免。太常宗俱爲司空，前司空許栩爲司徒。

<a id="hhs.8.p47"></a>
冬，鮮卑寇并州。

<a id="hhs.8.p48"></a>
熹平元年春三月壬戌，太傅胡廣薨。

<a id="hhs.8.p49"></a>
夏五月己巳，大赦天下，改元熹平。

<a id="hhs.8.p50"></a>
長樂太僕侯覽有罪，自殺。

<a id="hhs.8.p51"></a>
六月，京師雨水。

<a id="hhs.8.p52"></a>
癸巳，皇太后竇氏崩。秋七月甲寅，葬桓思皇后。

<a id="hhs.8.p53"></a>
宦官諷司隷校尉段熲捕繫太學諸生千餘人。冬十月，渤海王悝被誣謀反，丁亥，悝及妻子皆自殺。

<a id="hhs.8.p54"></a>
十一月，會稽人許生自稱「越王」，寇郡縣，遣楊州刺史臧旻、丹陽太守陳夤討破之。

<a id="hhs.8.p55"></a>
十二月，司徒許栩罷，大鴻臚袁隗爲司徒。

<a id="hhs.8.p56"></a>
鮮卑寇并州。

<a id="hhs.8.p57"></a>
是歲，甘陵王恢薨。

<a id="hhs.8.p58"></a>
二年春正月，大疫，使使者巡行致醫藥。

<a id="hhs.8.p59"></a>
丁丑，司空宗俱薨。

<a id="hhs.8.p60"></a>
二月壬午，大赦天下。

<a id="hhs.8.p61"></a>
以光祿勳楊賜爲司空。

<a id="hhs.8.p62"></a>
三月，太尉李咸免。夏五月，以司隷校尉段熲爲太尉。

<a id="hhs.8.p63"></a>
沛相師遷坐誣罔國王，下獄死。

<a id="hhs.8.p64"></a>
六月，北海地震。東萊，北海海水溢。

<a id="hhs.8.p65"></a>
秋七月，司空楊賜免，太常潁川唐珍爲司空。

<a id="hhs.8.p66"></a>
冬十二月，日南徼外國重譯貢獻。

<a id="hhs.8.p67"></a>
太尉段熲罷。

<a id="hhs.8.p68"></a>
鮮卑寇幽并二州。

<a id="hhs.8.p69"></a>
癸酉晦，日有食之。

<a id="hhs.8.p70"></a>
三年春正月，夫餘國遣使貢獻。

<a id="hhs.8.p71"></a>
二月己巳，大赦天下。

<a id="hhs.8.p72"></a>
太常陳耽爲太尉。

<a id="hhs.8.p73"></a>
三月，中山王暢薨，無子，國除。

<a id="hhs.8.p74"></a>
夏六月，封河閒王利子康爲濟南王，奉孝仁皇祀。

<a id="hhs.8.p75"></a>
冬十月癸丑，令天下繫囚罪未決，入縑贖。

<a id="hhs.8.p76"></a>
十一月，楊州刺史臧旻率丹陽太守陳寅，大破許生於會稽，斬之。

<a id="hhs.8.p77"></a>
任城王博薨。

<a id="hhs.8.p78"></a>
十二月，鮮卑寇北地，北地太守夏育追擊破之。鮮卑又寇并州。

<a id="hhs.8.p79"></a>
司空唐珍罷，永樂少府許訓爲司空。

<a id="hhs.8.p80"></a>
四年春三月，詔諸儒正五經文字，刻石立于太學門外。

<a id="hhs.8.p81"></a>
封河閒王建孫佗爲任城王。

<a id="hhs.8.p82"></a>
夏四月，郡國七大水。

<a id="hhs.8.p83"></a>
五月丁卯，大赦天下。

<a id="hhs.8.p84"></a>
延陵園灾，遣使者持節告祠延陵。

<a id="hhs.8.p85"></a>
鮮卑寇幽州。

<a id="hhs.8.p86"></a>
六月，弘農、三輔螟。

<a id="hhs.8.p87"></a>
遣守宮令之鹽監，穿渠爲民興利。

<a id="hhs.8.p88"></a>
令郡國遇災者，減田租之半；其傷害十四以上，勿收責。

<a id="hhs.8.p89"></a>
冬十月丁巳，令天下繫囚罪未決，入縑贖。

<a id="hhs.8.p90"></a>
拜沖帝母虞美人爲憲園貴人，質帝母陳夫人爲渤海孝王妃。

<a id="hhs.8.p91"></a>
改平準爲中準，使宦者爲令，列於內署。自是諸署悉以閹人爲丞、令。

<a id="hhs.8.p92"></a>
五年夏四月癸亥，大赦天下。

<a id="hhs.8.p93"></a>
益州郡夷叛，太守李顒討平之。

<a id="hhs.8.p94"></a>
復崇高山名爲嵩高山。

<a id="hhs.8.p95"></a>
大雩。使侍御史行詔獄亭部，理冤枉，原輕繫，休囚徒。

<a id="hhs.8.p96"></a>
五月，太尉陳耽罷，司空許訓爲太尉。

<a id="hhs.8.p97"></a>
閏月，永昌太守曹鸞坐訟黨人，弃市。詔黨人門生故吏父兄子弟在位者，皆免官禁錮。

<a id="hhs.8.p98"></a>
六月壬戌，太常南陽劉逸爲司空。

<a id="hhs.8.p99"></a>
秋七月，太尉許訓罷，光祿勳劉寬爲太尉。

<a id="hhs.8.p100"></a>
冬十月壬午，御殿後槐樹自拔倒豎。

<a id="hhs.8.p101"></a>
司徒袁隗罷。十一月丙戌，光祿大夫楊賜爲司徒。

<a id="hhs.8.p102"></a>
十二月，甘陵王定薨。

<a id="hhs.8.p103"></a>
試太學生年六十以上百餘人，除郎中、太子舍人至王家郎、郡國文學吏。

<a id="hhs.8.p104"></a>
是歲，鮮卑寇幽州。沛國言黃龍見譙。

<a id="hhs.8.p105"></a>
六年春正月辛丑，大赦天下。

<a id="hhs.8.p106"></a>
二月，南宮平城門及武庫東垣屋自壞。

<a id="hhs.8.p107"></a>
夏四月，大旱，七州蝗。

<a id="hhs.8.p108"></a>
鮮卑寇三邊。

<a id="hhs.8.p109"></a>
市賈民爲宣陵孝子者數十人，皆除太子舍人。

<a id="hhs.8.p110"></a>
秋七月，司空劉逸免，衞尉陳球爲司空。

<a id="hhs.8.p111"></a>
八月，遣破鮮卑中郎將田晏出雲中，使匈奴中郎將臧旻與南單于出鴈門，護烏桓校尉夏育出高柳，並伐鮮卑，晏等大敗。

<a id="hhs.8.p112"></a>
冬十月癸丑朔，日有食之。

<a id="hhs.8.p113"></a>
太尉劉寬免。

<a id="hhs.8.p114"></a>
辛丑，京師地震。

<a id="hhs.8.p115"></a>
辛亥，令天下繫囚罪未決，入縑贖。

<a id="hhs.8.p116"></a>
十一月，司空陳球免。十二月甲寅，太常河南孟戫爲太尉。庚辰，司徒楊賜免。太常陳耽爲司空。

<a id="hhs.8.p117"></a>
鮮卑寇遼西。

<a id="hhs.8.p118"></a>
永安太僕王旻下獄死。

<a id="hhs.8.p119"></a>
光和元年春正月，合浦、交阯烏滸蠻叛，招引九真、日南民攻沒郡縣。

<a id="hhs.8.p120"></a>
太尉孟戫罷。

<a id="hhs.8.p121"></a>
二月辛亥朔，日有食之。

<a id="hhs.8.p122"></a>
癸丑，光祿勳陳國袁滂爲司徒。

<a id="hhs.8.p123"></a>
始置鴻都門學生。

<a id="hhs.8.p124"></a>
三月辛丑，大赦天下，改元光和。

<a id="hhs.8.p125"></a>
太常常山張顥爲太尉。

<a id="hhs.8.p126"></a>
夏四月丙辰，地震。

<a id="hhs.8.p127"></a>
侍中寺雌雞化爲雄。

<a id="hhs.8.p128"></a>
司空陳耽免，太常來豔爲司空。

<a id="hhs.8.p129"></a>
五月壬午，有白衣人入德陽殿門，亡去不獲。六月丁丑，有黑氣墯所御溫德殿庭中。秋七月壬子，青虹見御坐玉堂後殿庭中。八月，有星孛于天巿。

<a id="hhs.8.p130"></a>
九月，太尉張顥罷，太常陳球爲太尉。司空來豔薨。冬十月，屯騎校尉袁逢爲司空。

<a id="hhs.8.p131"></a>
皇后宋氏廢，后父執金吾酆下獄死。

<a id="hhs.8.p132"></a>
丙子晦，日有食之。

<a id="hhs.8.p133"></a>
十一月，太尉陳球免。十二月丁巳，光祿大夫橋玄爲太尉。

<a id="hhs.8.p134"></a>
是歲，鮮卑寇酒泉。京師馬生人。初開西邸賣官，自關內侯、虎賁、羽林，入錢各有差。私令左右賣公卿，公千萬，卿五百萬。

<a id="hhs.8.p135"></a>
二年春，大疫，使常侍、中謁者巡行致醫藥。

<a id="hhs.8.p136"></a>
三月，司徒袁滂免，大鴻臚劉郃爲司徒。乙丑，太尉橋玄罷，太中大夫段熲爲太尉。

<a id="hhs.8.p137"></a>
司空袁逢罷，太常張濟爲司空。

<a id="hhs.8.p138"></a>
夏四月甲戌朔，日有食之。

<a id="hhs.8.p139"></a>
辛巳，中常侍王甫及太尉段熲並下獄死。

<a id="hhs.8.p140"></a>
丁酉，大赦天下，諸黨人禁錮小功以下皆除之。

<a id="hhs.8.p141"></a>
東平王端薨。

<a id="hhs.8.p142"></a>
五月，衞尉劉寬爲太尉。

<a id="hhs.8.p143"></a>
秋七月，使匈奴中郎將張脩有罪，下獄死。

<a id="hhs.8.p144"></a>
冬十月甲申，司徒劉郃、永樂少府陳球、衞尉陽球、步兵校尉劉納謀誅宦者，事泄，皆下獄死。

<a id="hhs.8.p145"></a>
巴郡板楯蠻叛，遣御史中丞蕭瑗督益州刺史討之，不剋。

<a id="hhs.8.p146"></a>
十二月，光祿勳楊賜爲司徒。

<a id="hhs.8.p147"></a>
鮮卑寇幽并二州。

<a id="hhs.8.p148"></a>
是歲，河閒王利薨。洛陽女子生兒，兩頭四臂。

<a id="hhs.8.p149"></a>
三年春正月癸酉，大赦天下。

<a id="hhs.8.p150"></a>
二月，公府駐駕廡自壞。

<a id="hhs.8.p151"></a>
三月，梁王元薨。

<a id="hhs.8.p152"></a>
夏四月，江夏蠻叛。

<a id="hhs.8.p153"></a>
六月，詔公卿舉能通尚書、毛詩、左氏、穀梁春秋各一人，悉除議郎。

<a id="hhs.8.p154"></a>
秋，表是地震，涌水出。

<a id="hhs.8.p155"></a>
八月，令繫囚罪未決，入縑贖，各有差。

<a id="hhs.8.p187"></a>

<a id="hhs.8.p190"></a>

<a id="hhs.8.p191"></a>
五月，皇甫嵩、朱儁復與波才等戰於長社，大破之。

<a id="hhs.8.p192"></a>
六月，南陽太守秦頡擊張曼成，斬之。

<a id="hhs.8.p193"></a>
交阯屯兵執刺史及合浦太守來達，自稱「柱天將軍」，遣交阯刺史賈琮討平之。

<a id="hhs.8.p194"></a>

<a id="hhs.8.p195"></a>
洛陽女子生兒，兩頭共身。

<a id="hhs.8.p196"></a>
秋七月，巴郡妖巫張脩反，寇郡縣。

<a id="hhs.8.p197"></a>
河南尹徐灌下獄死。

<a id="hhs.8.p198"></a>
八月，皇甫嵩與黃巾戰於倉亭，獲其帥。

<a id="hhs.8.p199"></a>
乙巳，詔皇甫嵩北討張角。

<a id="hhs.8.p200"></a>
九月，安平王續有罪誅，國除。

<a id="hhs.8.p201"></a>
冬十月，皇甫嵩與黃巾賊戰於廣宗，獲張角弟梁。角先死，乃戮其屍。以皇甫嵩爲左車騎將軍。十一月，皇甫嵩又破黃巾于下曲陽，斬張角弟寶。

<a id="hhs.8.p202"></a>
湟中義從胡北宮伯玉與先零羌叛，以金城人邊章、韓遂爲軍帥，攻殺護羌校尉伶徵、金城太守陳懿。

<a id="hhs.8.p203"></a>
癸巳，朱儁拔宛城，斬黃巾別帥孫夏。

<a id="hhs.8.p204"></a>
詔減太官珍羞，御食一肉；廄馬非郊祭之用，悉出給軍。

<a id="hhs.8.p205"></a>
十二月己巳，大赦天下，改元中平。

<a id="hhs.8.p206"></a>
是歲，下邳王意薨，無子，國除。郡國生異草，備龍蛇鳥獸之形。

<a id="hhs.8.p207"></a>
二年春正月，大疫。

<a id="hhs.8.p208"></a>
琅邪王據薨。

稅天下田，畒十錢。

黑山賊張牛角等十餘輩並起，所在寇鈔。

司徒袁隗免。三月，廷尉崔烈爲司徒。

<a id="hhs.8.p213"></a>
北宮伯玉等寇三輔，遣左車騎將軍皇甫嵩討之，不剋。

<a id="hhs.8.p214"></a>
夏四月庚戌，大風，雨雹。

五月，太尉鄧盛罷，太僕河南張延爲太尉。

<a id="hhs.8.p216"></a>
秋七月，三輔螟。

<a id="hhs.8.p217"></a>

<a id="hhs.8.p218"></a>
前司徒陳耽、諫議大夫劉陶坐直言，下獄死。

<a id="hhs.8.p242"></a>

<a id="hhs.8.p243"></a>
十一月，太尉崔烈罷，大司農曹嵩爲太尉。

<a id="hhs.8.p244"></a>
十二月，休屠各胡叛。

<a id="hhs.8.p245"></a>

<a id="hhs.8.p246"></a>
五年春正月，休屠各胡寇西河，殺郡守邢紀。

<a id="hhs.8.p247"></a>
丁酉，大赦天下。

<a id="hhs.8.p248"></a>
二月，有星孛于紫宮。

<a id="hhs.8.p249"></a>
黃巾餘賊郭大等起於西河白波谷，寇太原、河東。

<a id="hhs.8.p250"></a>
三月，休屠各胡攻殺并州刺史張懿，遂與南匈奴左部胡合，殺其單于。

<a id="hhs.8.p251"></a>
夏四月，汝南葛陂黃巾攻沒郡縣。

<a id="hhs.8.p252"></a>
太尉曹嵩罷。五月，永樂少府樊陵爲太尉。

<a id="hhs.8.p253"></a>
六月丙寅，大風。

<a id="hhs.8.p254"></a>
太尉樊陵罷。

<a id="hhs.8.p255"></a>
益州黃巾馬相攻殺刺史郗儉，自稱天子，又寇巴郡，殺郡守趙部，益州從事賈龍擊相，斬之。

<a id="hhs.8.p256"></a>
郡國七大水。

<a id="hhs.8.p257"></a>
秋七月，射聲校尉馬日磾爲太尉。

<a id="hhs.8.p258"></a>
八月，初置西園八校尉。

<a id="hhs.8.p259"></a>
司徒許相罷，司空丁宮爲司徒。光祿勳南陽劉弘爲司空。衞尉董重爲票騎將軍。

<a id="hhs.8.p260"></a>
九月，南單于叛，與白波賊寇河東。遣中郎將孟益率騎都尉公孫瓉討漁陽賊張純等。

<a id="hhs.8.p261"></a>
冬十月，壬午，御殿後槐樹自拔倒豎。青、徐黃巾復起，寇郡縣。

<a id="hhs.8.p262"></a>
甲子，帝自稱「無上將軍」，燿兵於平樂觀。

<a id="hhs.8.p263"></a>
十一月，涼州賊王國圍陳倉，右將軍皇甫嵩救之。

<a id="hhs.8.p264"></a>
遣下軍校尉鮑鴻討葛陂黃巾。

<a id="hhs.8.p265"></a>
巴郡板楯蠻叛，遣上軍別部司馬趙瑾討平之。

<a id="hhs.8.p266"></a>
公孫瓉與張純戰於石門，大破之。

<a id="hhs.8.p267"></a>
是歲，改刺史，新置牧。

<a id="hhs.8.p268"></a>
六年春二月，左將軍皇甫嵩大破王國於陳倉。

<a id="hhs.8.p269"></a>
三月，幽州牧劉虞購斬漁陽賊張純。

<a id="hhs.8.p270"></a>
下軍校尉鮑鴻下獄死。

<a id="hhs.8.p271"></a>
夏四月丙午朔，日有食之。

<a id="hhs.8.p272"></a>
太尉馬日磾免，幽州牧劉虞爲太尉。

<a id="hhs.8.p273"></a>
丙辰，帝崩于南宮嘉德殿，年三十四。戊午，皇子辯即皇帝位，年十七。尊皇后曰皇太后，太后臨朝。大赦天下，改元爲光喜。封皇弟協爲渤海王。後將軍袁隗爲太傅，與大將軍何進參錄尚書事。上軍校尉蹇碩下獄死。五月辛巳，票騎將軍董重下獄死。六月辛亥，孝仁皇后董氏崩。

<a id="hhs.8.p274"></a>
辛酉，葬孝靈皇帝于文陵。

<a id="hhs.8.p275"></a>
秋七月，甘陵王忠薨。

<a id="hhs.8.p276"></a>
庚寅，孝仁皇后歸葬河閒慎陵。

<a id="hhs.8.p277"></a>
徙渤海王協爲陳留王。司徒丁宮罷。

<a id="hhs.8.p278"></a>
八月戊辰，中常侍張讓、段珪等殺大將軍何進，於是虎賁中郎將袁術燒東西宮，攻諸宦者。庚午，張讓、段珪等劫少帝及陳留王幸北宮德陽殿。何進部曲將吳匡與車騎將軍何苗戰於朱雀闕下，苗敗斬之。辛未，司隷校尉袁紹勒兵收僞司隷校尉樊陵、河南尹許相及諸閹人，無少長皆斬之。讓、珪等復劫少帝、陳留王走小平津。尚書盧植追讓、珪等，斬數人，其餘投河而死。帝與陳留王協夜步逐熒光行數里，得民家露車，共乘之。

<a id="hhs.8.p279"></a>
辛未，還宮。大赦天下，改光喜爲昭寧。

<a id="hhs.8.p280"></a>
并州牧董卓殺執金吾丁原。司空劉弘免，董卓自爲司空。

<a id="hhs.8.p281"></a>
九月甲戌，董卓廢帝爲弘農王。

<a id="hhs.8.p282"></a>
自六月雨，至于是月。

<a id="hhs.8.p283"></a>
論曰：秦本紀說趙高譎二世，指鹿爲馬，而趙忠、張讓亦紿靈帝不得登高臨觀，故知亡敝者同其致矣。然則靈帝之爲靈也優哉！

<a id="hhs.8.p284"></a>
贊曰：靈帝負乘，委體宦孽。徵亡備兆，小雅盡缺。麋鹿霜露，遂棲宮衞。
